# crisp-eda
Repository  showcasing a power BI project
ASSIGNMENT:
The objective is to analyse a dataset from Kaggle showing different YouTube short contents. Explore and derive insights from the data that shows the engagement levels from different categories; to see how different categories perform in terms of engagement
Parameters that will be included:
1.	Sum of views: to see how many viewers the content reaches.
2. Sun of duration: to determine if the duration of the short influences the engagement.
3. Type of content: to discover the engagement of each content.
4. 	Sum of likes: to see how each content is received
5. 	Sum of shares: to determine the engagement of each content
